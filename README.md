# EventoAcademicoApp

## Descripción breve
EventoAcademicoApp es un sistema de registro para el Congreso Estudiantil de Innovación TI 2026.  
Permite registrar participantes mediante un formulario web, validar los datos ingresados y mostrar un resumen de los inscritos en la sesión.

## Instrucciones básicas para ejecutar el proyecto
1. Abrir la solución del proyecto en Visual Studio.
2. Ejecutar la aplicación con IIS Express o el servidor configurado.
3. Ingresar los datos del formulario y completar el registro.
4. Revisar la página de resumen para ver los participantes registrados.
